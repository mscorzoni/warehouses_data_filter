# Warehouses data filter

## Installation

Go to client folder and install node module

```bash
  $ yarn install
```

Go back to root and run:

```bash
  $ rails db:setup
```

```bash
  $ foreman start -f Procfile.dev
```

### PORTS

* client: 3005
* server: 3004


### Version

* ruby 2.4.4
* Rails 5.2.1
